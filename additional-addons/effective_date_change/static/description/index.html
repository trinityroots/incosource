Change Effective Date

Effective date is a date at which the transfer has been processed or cancelled. Normally, when you validated a pickings (Receives or Delivery Order), current today's date will always be chosen as it's processed date and that will also affect stock journals calculation.

But, what if you want to choose another date instead?

Well, This Module is Your Solution

Allows you to change the effective date after a "Done" state.



Support Future or Past Date Changing

Allows you to set the date from past.



Not Only Changes Inside The Pickings

But also in Stock Moves and Journal, automatically!








Save Your Time

By changing multiple-related record easily, with just one click!



For selectable user

Set certain user to have this privilege of changing effective date



Demonstration Video in Youtube

Installation Procedure

  1. Download this module and put it inside your odoo addons file.
  2. From Odoo interface, enter the debug mode > Apps menu > update apps.
  3. In the search bar, delete everything and type in Change Effective Date
  4. Restart your Odoo service
  5. Go to settings > Select User > Grant The Privilege
  6. Refresh your browser, now you will be able to set the effective date
